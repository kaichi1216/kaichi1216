heroku 更改table 出現錯誤訊息

2019-10-23

前後端開發 Ruby 0n Rails 菜鳥日誌

修改 table 欄位後 推至 heroku 出現錯誤訊息

有在本地端 新增 刪除 修改 欄位後 推上 heroku 都成功 但打開app後 跳出錯誤訊息 到 console 查看 是 資料欄位出錯 查了一下資料 原來當你有修改 table 資料後 要執行 restart 指令

再開啟網頁就正常囉!!
