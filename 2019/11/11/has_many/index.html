has_many :through

has_many :through 用在多對多關聯資料庫

以 醫院 舉例 一個醫生 可以有很多病人，一個病人可能全身都有病 所以他可以掛很多不同科的醫生
醫生跟病人都都各自是一個 model ，建立多對多的關係時，這時我們就會有第三個 model 去記錄他們的關係 第三個 model 先叫做 醫院 負責記錄每個病人的看診醫生跟醫生看診的病人的紀錄
